# Next.js React Assignment

This is a Next.js React assignment project.

### Prerequisites
- Node.js (v18+ recommended)
- npm

### Clone the Repository
```bash
git clone https://github.com/aminishere/frontend_assignment.git
cd frontend_assignment
```

### Install Dependencies
```
npm install
```

### Run the Development Server
```
npm run dev
```

Open the browser and go to:
```
http://localhost:3000/dashboard
```
### Run tests
```
npm test
```
